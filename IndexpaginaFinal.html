¿Qué es HTML?

HTML es el lenguaje base de la web.

Permite estructurar contenido como texto, imágenes y enlaces.

Es fundamental para cualquier página web.

Se combina con CSS y JavaScript.

HTML comenzó en los años 90

Estructura HTML5

HTML5 introdujo etiquetas como:

  • <header>
  • <nav>
  • <section>
  • <article>
  • <footer>

Esto ayuda a organizar mejor las páginas.
